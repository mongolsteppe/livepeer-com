# Usage and Billing

Login to the Livepeer.com Dashboard. Usage for the current billing cycle is
displayed on the Home page,
[livepeer.com/dashboard](https://livepeer.com//dashboard).

If you would like to upgrade your plan, view past invoices, update your payment
method, visit
[livepeer.com/dashboard/billing](https://livepeer.com/dashboard/billing).

If you'd like to view all the benefits of our different plans, visit
[livepeer.com/dashboard/plans](https://livepeer.com/pricing). If you
have additional questions about the plans or want to hear more, send us a note
at hello@livepeer.com or use our [contact form](https://livepeer.com/contact).

Livepeer.com does not currently charge for stream delivery via CDN or recording
storage. We will notify customers before we start to do so.
